====================================================
  EXPENSE TRACKER — Java Swing GUI Project
====================================================

FILES:
  src/
    Expense.java            --> Model class
    ExpenseTrackerGUI.java  --> Main GUI application

HOW TO RUN IN VS CODE:
------------------------
1. Open VS Code
2. Open folder: ExpenseTrackerGUI
3. Terminal > New Terminal  (Ctrl + `)
4. Type:   cd src
5. Type:   javac *.java
6. Type:   java ExpenseTrackerGUI

A WINDOW will open — this is the GUI! :)

FEATURES:
---------
- Add expenses with name, amount, category, date
- View all expenses in a table
- Search by name or category
- Delete selected expense (with confirmation)
- Edit selected expense
- Show total popup
- 7 sample expenses pre-loaded
