
# Expense Tracker Application (Java Swing)

## 📌 Overview

This is a desktop-based Expense Tracker application developed using Java Swing.
The application allows users to manage their daily expenses efficiently through a graphical user interface.

---

## 🚀 Features

* Add new expenses
* Edit existing expenses
* Delete selected expenses
* Search expenses by name or category
* View total expenses
* User-friendly GUI

---

## 🛠️ Technologies Used

* Java
* Java Swing (GUI)
* OOP Concepts
* ArrayList (for data storage)

---

## 📂 Project Structure

```
ExpenseTracker/
│
├── src/
│   ├── ExpenseTrackerGUI.java
│   └── Expense.java
```

---

## ▶️ How to Run

### Step 1: Compile

```
javac *.java
```

### Step 2: Run

```
java ExpenseTrackerGUI
```

---

## 💻 Requirements

* Java JDK 8 or above
* Any IDE (VS Code / IntelliJ / Eclipse)
